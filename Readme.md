/*遇到的问题*/

1 测试的时候,服务器和客户端都在本地,可以进行远程测试吗？

解：
	服务器在一个局域网,而客户端在另一个局域网,是可以进行Http测试;
例如,服务器在公网,可以将服务器托管到云服务商上，而客户端在任意网络下进行访问。
客户端可以通过服务器的公网ip或者域名进行访问;服要开放相关端口（80或者443）
如果使用https,需要配置SSL/TIS证书

2 http协议

解：
	(1)客户端(浏览器)向服务器发送请求
	(2)服务器对请求进行解析并处理
	(3)服务器对客户端(浏览器)的请求发送响应
	(4)客户端(浏览器)对响应进行解析并处理
