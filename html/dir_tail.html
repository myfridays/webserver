This project is a lightweight web server based on epoll, with the following main functions

  • Login function: This server is capable of processing login requests and performing basic username and password verification.
  • Starry Sky Background Animation: The login interface has a starry sky animation background, implemented through HTML5 canvas, making the login interface more dynamic and attractive.
  • HTTP request processing: The server is able to accept HTTP requests and process the corresponding logic based on the request path, such as handling POST requests for login forms.
  • Redirection mechanism: After successful login, the server will redirect the user to the homepage via HTTP to avoid duplicate submission of form data (using Post/Direct/Get mode).
  • File response function: The server supports sendi ng static files, such as HTML pages, images, and other resources, to the client
